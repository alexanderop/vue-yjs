# Installation

## Prerequisites

- Vue 3.3+
- Node.js 18+
- A package manager (pnpm recommended)

## Install

::code-group
```bash [pnpm]
pnpm add vue-yjs yjs
```
```bash [npm]
npm install vue-yjs yjs
```
```bash [yarn]
yarn add vue-yjs yjs
```
::

## Optional Dependencies

vue-yjs has optional peer dependencies for specific features:

| Package | Required for |
|---------|-------------|
| `y-websocket` | `useWebSocketProvider`, `useYRoom` |
| `y-protocols` | `useAwareness` |
| `y-indexeddb` | `useIndexedDB`, `useYRoom` with `persist: true` |

Install only what you need:

```bash
pnpm add y-websocket y-protocols y-indexeddb
```

## TypeScript

vue-yjs is written in TypeScript and ships type declarations. No `@types/` packages needed.
