# useY

Reactive binding for any Yjs shared type. Returns a readonly shallow ref that stays in sync with the underlying CRDT via `observeDeep`. The observer is cleaned up automatically when the enclosing effect scope is disposed.

::note
`useY` is the low-level building block. For typed operations (set, push, delete), prefer [`useYMap`](/composables/shared-types/use-y-map), [`useYArray`](/composables/shared-types/use-y-array), or [`useYText`](/composables/shared-types/use-y-text).
::

## Usage

```vue
<script setup lang="ts">
import { useY } from 'vue-yjs'
import * as Y from 'yjs'

const doc = new Y.Doc()
const yArray = doc.getArray<string>('items')

// Type is automatically inferred: Ref<string[]>
const items = useY(yArray)
</script>

<template>
  <div v-for="item in items" :key="item">{{ item }}</div>
</template>
```

With explicit type override:

```ts
interface Todo { id: string; text: string; done: boolean }

const yTodos = doc.getArray<Y.Map<unknown>>('todos')
const todos = useY<Todo[]>(yTodos) // Ref<Todo[]>
```

## Parameters

::field-group
  ::field{name="yData" type="Y.AbstractType" required}
  The Yjs shared type to observe. Works with `Y.Array`, `Y.Map`, `Y.Text`, `Y.XmlFragment`, and nested structures.
  ::
::

## Return Value

::field-group
  ::field{name="Returns"}
  `Readonly<ShallowRef<T>>` — A readonly shallow ref containing the JSON snapshot of the Yjs type. Updates whenever `observeDeep` fires and the data has actually changed.
  ::
::

## Change Detection

Uses `equalityDeep` from `lib0` (Yjs's utility library) to avoid unnecessary reactivity triggers when the data hasn't actually changed. This means Vue only re-renders when the data is structurally different.

## Type Inference

`useY` supports two call signatures:

1. **Auto-inferred** — The return type is derived from the Yjs type using the `YTypeToJson` mapped type:
   - `Y.Array<V>` → `Array<YTypeToJson<V>>`
   - `Y.Map<V>` → `{ [key: string]: YTypeToJson<V> }`
   - `Y.Text | Y.XmlText | Y.XmlFragment` → `string`

2. **Explicit override** — Pass a type parameter to override inference:
   ```ts
   const data = useY<MyCustomType>(yType)
   ```

## Type Declarations

::collapsible{label="Show type declarations"}
```ts
type YTypeToJson<YType> =
  YType extends Y.Array<infer Value>
    ? Array<YTypeToJson<Value>>
    : YType extends Y.Map<infer MapValue>
      ? { [key: string]: YTypeToJson<MapValue> }
      : YType extends Y.XmlFragment | Y.XmlText | Y.Text
        ? string
        : YType

declare function useY<TJson>(
  yData: Y.AbstractType<any>,
): Readonly<ShallowRef<TJson>>

declare function useY<YType extends Y.AbstractType<any>>(
  yData: YType,
): Readonly<ShallowRef<YTypeToJson<YType>>>
```
