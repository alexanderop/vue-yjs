# useYMap

Typed reactive binding for a `Y.Map` shared type. Injects the `Y.Doc` from the component tree, creates or retrieves a named map, and returns reactive data with mutation helpers.

::tip
Use the `defaults` option to seed initial values. Defaults are only applied when the map is empty — persisted or synced data always takes precedence.
::

## Usage

```vue
<script setup lang="ts">
import { useYMap } from 'vue-yjs'

interface Settings {
  theme: string
  fontSize: number
}

const { data, set, delete: del, yMap } = useYMap<Settings>('settings', {
  defaults: { theme: 'dark', fontSize: 14 }
})
</script>

<template>
  <p>Theme: {{ data.theme }}</p>
  <button @click="set('fontSize', data.fontSize + 1)">
    Increase font size ({{ data.fontSize }})
  </button>
</template>
```

## Parameters

::field-group
  ::field{name="name" type="string" required}
  Name of the shared map in the `Y.Doc`.
  ::

  ::field{name="options"}
  `{ defaults?: Partial<T> }` — Initial values applied only when the map is empty (won't overwrite persisted or synced data).
  ::
::

## Return Value

::field-group
  ::field{name="data"}
  `Readonly<ShallowRef<T>>` — Reactive snapshot of the map's contents.
  ::

  ::field{name="set"}
  `<K extends keyof T & string>(key: K, value: T[K]) => void` — Set a key-value pair. Nested objects/arrays are automatically converted to `Y.Map`/`Y.Array` via `toYType`.
  ::

  ::field{name="delete"}
  `(key: keyof T & string) => void` — Delete a key from the map.
  ::

  ::field{name="yMap" type="Y.Map"}
  The underlying Yjs map instance for advanced operations.
  ::
::

## Defaults Behavior

Defaults are applied inside a `doc.transact()` only when `yMap.size === 0`. This means:

- Defaults populate the map on first use
- Persisted data (from IndexedDB) or synced data (from WebSocket) takes precedence
- Nested objects in defaults are converted to Yjs types via `toYType`

## Type Declarations

::collapsible{label="Show type declarations"}
```ts
interface UseYMapReturn<T extends { [K in keyof T]: unknown }> {
  data: Readonly<ShallowRef<T>>
  set: <K extends keyof T & string>(key: K, value: T[K]) => void
  delete: (key: keyof T & string) => void
  yMap: Y.Map<unknown>
}

declare function useYMap<T extends { [K in keyof T]: unknown }>(
  name: string,
  options?: { defaults?: Partial<T> },
): UseYMapReturn<T>
```
