# useYArray

Typed reactive binding for a `Y.Array` shared type. Injects the `Y.Doc` from the component tree, creates or retrieves a named array, and returns reactive items with mutation helpers.

::warning
The `update`, `findIndex`, `updateBy`, and `deleteBy` methods only work with arrays of objects (backed by `Y.Map`). Calling them on an array of primitives will throw an error.
::

## Usage

```vue
<script setup lang="ts">
import { useYArray } from 'vue-yjs'

interface Todo {
  id: string
  text: string
  done: boolean
}

const { items, push, update, delete: del, yArray } = useYArray<Todo>('todos')
</script>

<template>
  <ul>
    <li v-for="(todo, i) in items" :key="todo.id">
      <input
        type="checkbox"
        :checked="todo.done"
        @change="update(i, { done: !todo.done })"
      />
      {{ todo.text }}
      <button @click="del(i)">Remove</button>
    </li>
  </ul>
  <button @click="push({ id: crypto.randomUUID(), text: 'New todo', done: false })">
    Add Todo
  </button>
</template>
```

  ::field{name="name" type="string" required}
  Name of the shared array in the `Y.Doc`.
  ::

## Return Value

::field-group
  ::field{name="items"}
  `Readonly<ShallowRef<T[]>>` — Reactive snapshot of the array's contents.
  ::

  ::field{name="push"}
  `(...items: T[]) => void` — Append items to the end. Objects/arrays are auto-converted to Yjs types.
  ::

  ::field{name="insert"}
  `(index: number, ...items: T[]) => void` — Insert items at a specific index.
  ::

  ::field{name="update"}
  `(index: number, partial: Partial<T>) => void` — Partially update the item at the given index. Only works with `Y.Map` items (arrays of objects).
  ::

  ::field{name="delete"}
  `(index: number, count?: number) => void` — Delete `count` items starting at `index` (default: 1).
  ::

  ::field{name="findIndex"}
  `<K extends keyof T & string>(key: K, value: T[K]) => number` — Find the index of the first item where `item[key] === value`. Returns `-1` if not found. Only works with `Y.Map` items.
  ::

  ::field{name="updateBy"}
  `<K extends keyof T & string>(key: K, value: T[K], partial: Partial<T>) => boolean` — Find the first item where `item[key] === value` and partially update it. Returns `true` if found, `false` otherwise.
  ::

  ::field{name="deleteBy"}
  `<K extends keyof T & string>(key: K, value: T[K]) => boolean` — Find the first item where `item[key] === value` and delete it. Returns `true` if found, `false` otherwise.
  ::

  ::field{name="yArray" type="Y.Array"}
  The underlying Yjs array instance for advanced operations.
  ::
::

## Update Behavior

The `update` method uses `doc.transact()` to batch multiple field changes into a single operation. It only works with array items that are `Y.Map` instances (i.e., arrays of objects). Calling `update` on an array of primitives throws an error.

## ID-Based Operations

In concurrent CRDT editing, array indices shift unpredictably. The `findIndex`, `updateBy`, and `deleteBy` methods let you locate items by a property value (typically `id`) instead of relying on indices.

```ts
const { items, updateBy, deleteBy, findIndex, update } = useYArray<Todo>('todos')

// Delete by id — no need to find the index first
deleteBy('id', todoId)

// Update by id
updateBy('id', todoId, { text: 'Updated text' })

// When you need the current value (e.g., toggling a boolean), use findIndex + update
const idx = findIndex('id', todoId)
if (idx !== -1) {
  update(idx, { done: !items.value[idx]!.done })
}
```

All three methods return `-1` or `false` when the item is not found, which is normal during concurrent editing when another user may have already deleted the item.

## Type Declarations

::collapsible{label="Show type declarations"}
```ts
interface UseYArrayReturn<T> {
  items: Readonly<ShallowRef<T[]>>
  push: (...items: T[]) => void
  insert: (index: number, ...items: T[]) => void
  update: (index: number, partial: Partial<T>) => void
  delete: (index: number, count?: number) => void
  findIndex: <K extends keyof T & string>(key: K, value: T[K]) => number
  updateBy: <K extends keyof T & string>(key: K, value: T[K], partial: Partial<T>) => boolean
  deleteBy: <K extends keyof T & string>(key: K, value: T[K]) => boolean
  yArray: Y.Array<unknown>
}

declare function useYArray<T>(name: string): UseYArrayReturn<T>
```
