# useYText

Typed reactive binding for a `Y.Text` shared type. Injects the `Y.Doc` from the component tree, creates or retrieves a named text, and returns a reactive string with mutation helpers.

::note
For real-time collaborative editing, consider integrating with a rich text editor that supports Yjs bindings (e.g., [TipTap](https://tiptap.dev), [ProseMirror](https://prosemirror.net), [Monaco](https://microsoft.github.io/monaco-editor/)) rather than using `replace` directly.
::

## Usage

```vue
<script setup lang="ts">
import { useYText } from 'vue-yjs'

const { text, replace, yText } = useYText('document')
</script>

<template>
  <textarea
    :value="text"
    @input="replace(($event.target as HTMLTextAreaElement).value)"
  />
  <p>{{ text.length }} characters</p>
</template>
```

## Parameters

::field-group
  ::field{name="name" type="string" required}
  Name of the shared text in the `Y.Doc`.
  ::
::

## Return Value

::field-group
  ::field{name="text"}
  `Readonly<ShallowRef<string>>` — Reactive string of the text contents.
  ::

  ::field{name="insert"}
  `(index: number, content: string) => void` — Insert text at the given position.
  ::

  ::field{name="delete"}
  `(index: number, length: number) => void` — Delete `length` characters starting at `index`.
  ::

  ::field{name="replace"}
  `(content: string) => void` — Replace the entire text content. Uses `doc.transact()` for an atomic delete + insert (single undo step).
  ::

  ::field{name="yText" type="Y.Text"}
  The underlying Yjs text instance for advanced operations (formatting, embeds).
  ::
::

## Replace Behavior

The `replace` method performs an atomic delete-all + insert inside `doc.transact()`. This means:

- The entire operation counts as a single undo step
- Concurrent edits from other clients are handled correctly by Yjs's CRDT algorithm

For real-time collaborative text editing, consider using `insert` and `delete` for finer-grained operations, or integrate with a text editor that supports Yjs bindings (e.g., TipTap, ProseMirror, Monaco).
